3. HASIL DAN PEMBAHASAN

   3.1 IMPLEMENTASI SISTEM
   -----------------------
       
       A. DESKRIPSI STRUKTUR PROYEK DAN ARSITEKTUR SISTEM
       
       Proyek ini mengembangkan aplikasi web berbasis Flask untuk implementasi, 
       analisis, dan visualisasi algoritma Advanced Encryption Standard (AES). 
       Aplikasi dibangun dengan arsitektur client-server yang memisahkan logic 
       kriptografi di backend dengan interface interaktif di frontend.
       
       Struktur proyek terdiri atas beberapa komponen utama:
       
       1. Backend (Server-side):
          - Menggunakan Flask sebagai framework web Python
          - Menyediakan REST API endpoints untuk operasi enkripsi/dekripsi
          - Menangani pemrosesan data dan analisis S-Box
          - Memvalidasi input dan mengelola keamanan endpoint
       
       2. Frontend (Client-side):
          - Menggunakan HTML5, CSS3, dan JavaScript vanilla
          - Menyediakan interface interaktif dengan multiple tabs/pages
          - Real-time visualization dan output rendering
          - Responsive design untuk berbagai resolusi layar
       
       3. Struktur file dan folder:
          ├── Backend Modules (aes_cipher.py, sbox_analyzer.py, app.py)
          ├── Testing (audit_tests.py)
          ├── Frontend (templates/index.html, static/*)
          └── Configuration (requirements.txt, Procfile, runtime.txt)
       
       
       B. MODUL-MODUL YANG TELAH DIKEMBANGKAN
       
       1. aes_cipher.py - Implementasi Algoritma AES
          
          Modul ini mengimplementasikan seluruh algoritma Advanced Encryption 
          Standard sesuai dengan standar FIPS 197. Komponen utama meliputi:
          
          • Class AESCipher: Kelas utama yang menangani enkripsi dan dekripsi
            - Inisialisasi dengan kunci (128, 192, atau 256 bit)
            - Support untuk custom S-Box (default menggunakan AES S-Box standar)
          
          • Key Expansion (key_expansion):
            - Mengimplementasikan Rcon (Round constants)
            - RotWord dan SubWord operations
            - Ekspansi kunci menjadi round keys (10, 12, atau 14 round)
          
          • Operasi Enkripsi:
            - SubBytes: Substitusi menggunakan S-Box pada setiap byte state
            - ShiftRows: Pergeseran baris dengan offset (0,1,2,3 untuk setiap baris)
            - MixColumns: Perkalian matriks di Galois Field GF(2^8)
            - AddRoundKey: XOR antara state dengan round key
          
          • Operasi Dekripsi:
            - Inverse SubBytes (InvSubBytes) menggunakan Inverse S-Box
            - Inverse ShiftRows (InvShiftRows)
            - Inverse MixColumns (InvMixColumns)
            - AddRoundKey (sama dengan enkripsi)
          
          • Operasi Pendukung:
            - GF(2^8) multiplication untuk MixColumns
            - State matrix management (4x4 byte matrix)
            - Round key extraction
       
       
       2. app.py - Backend Aplikasi Web dengan Flask
          
          Modul ini merupakan core dari web application yang menyediakan endpoints
          API untuk berbagai fitur. Fitur-fitur utama:
          
          • Routes dan Endpoints:
            - /: Main page rendering
            - /api/encrypt-text: Endpoint untuk enkripsi plaintext
            - /api/decrypt-text: Endpoint untuk dekripsi ciphertext
            - /api/analyze-sbox: Analisis karakteristik S-Box
            - /api/get-sbox-matrix: Retrieve S-Box matrix
            - /api/compare-sboxes: Perbandingan antar S-Box
            - /api/image-encrypt: Enkripsi image data
            - /api/generate-report: Generate laporan hasil analisis
          
          • Fitur Enkripsi/Dekripsi:
            - Support untuk plaintext dengan padding PKCS7
            - Support multiple key sizes (128, 192, 256 bit)
            - Output dalam format hexadecimal dan base64
            - Error handling untuk invalid input
          
          • Fitur Analisis S-Box:
            - Kalkulasi statistik keamanan (nonlinearity, differential uniformity)
            - Linear Approximation Table (LAT) dan Difference Distribution Table (DDT)
            - Visualisasi matriks S-Box
            - Export hasil analisis ke Excel
          
          • Manajemen Kunci:
            - Parsing dari berbagai format input (text, hex)
            - Automatic padding/truncation ke ukuran yang tepat
            - Default key untuk testing
          
          • Security Features:
            - Input validation dan sanitization
            - Rate limiting (optional)
            - CORS handling untuk cross-origin requests
       
       
       3. sbox_analyzer.py - Analisis Karakteristik S-Box
          
          Modul ini mengimplementasikan analisis mendalam terhadap substitution box
          dan menyediakan tools untuk evaluasi keamanan. Fitur-fitur meliputi:
          
          • S-Box Repository:
            - AES S-Box standar (256 values)
            - SBOX_44 (dari penelitian yang direplikasi)
            - Support untuk custom S-Box
          
          • Analisis Keamanan S-Box:
            - check_bijective: Verifikasi sifat bijektif (permutasi)
            - check_balance: Cek keseimbangan output bits (0/1)
            - calculate_nonlinearity: Menghitung derajat non-linearity
            - calculate_sac: Strict Avalanche Criterion
            - calculate_differential_uniformity: Differential cryptanalysis resistance
            - calculate_algebraic_degree: Derajat algebraik
          
          • Cryptanalysis Tables:
            - get_lat_table: Membuat Linear Approximation Table
            - get_ddt_table: Membuat Difference Distribution Table
            - Analisis distribusi diferensial
          
          • Konstruksi S-Box:
            - construct_sbox_from_matrix: Konstruksi dari affine transformation
            - GF(2^8) operations (multiplication, inverse)
            - Implementation dari AES affine transformation
          
          • Visualisasi Data:
            - Matrix rendering untuk LAT dan DDT
            - Histogram untuk distribusi
            - Export data untuk reporting
       
       
       4. audit_tests.py - Testing dan Validasi
          
          Modul ini menyediakan comprehensive testing untuk validasi implementasi:
          
          • Unit Tests untuk Operasi Dasar:
            - test_gf_multiplication: Verifikasi GF(2^8) multiplication
            - test_gf_inverse: Verifikasi GF(2^8) inverse operations
            - test_aes_sbox_construction: Validasi S-Box construction
          
          • Functional Tests:
            - test_key_expansion: Verifikasi key schedule
            - test_sub_bytes: SubBytes operation validation
            - test_shift_rows: ShiftRows operation validation
            - test_mix_columns: MixColumns operation validation
            - test_add_round_key: AddRoundKey operation validation
          
          • Integration Tests:
            - test_encryption_decryption: End-to-end encryption/decryption
            - test_nist_vectors: Validasi dengan NIST test vectors
            - test_multiple_key_sizes: Testing AES-128, AES-192, AES-256
            - test_padding_and_unpadding: Padding validation
          
          • Security Tests:
            - Verifikasi keamanan S-Box terhadap known attacks
            - Testing terhadap weak keys
            - Performance benchmarking
